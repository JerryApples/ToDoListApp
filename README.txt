---- Οδηγίες Εγκατάστασης ----
https://github.com/JerryApples/ToDoListApp
Κατεβάστε το εκτελέσιμο αρχείο app.exe στον υπολογιστή σας.
Η εφαρμογή αναμένεται να δημιουργήσει ένα αρχείο στην τοποθεσία που 
θα το αποθηκεύσετε. Για αυτό προτείνετε η εγκατάσταση της εφαρμογής 
σε ξεχωριστό φάκελο.
Όταν τρέξετε το πρόγραμμα θα ανοίξει ένα παράθυρο με την εφαρμογή.


*****Αν έχετε κατεβάσει όλα τα αρχεία του κώδικα από άλλη πηγή****** 
Τρέξτε το εκτελέσιμο που βρίσκεται στον φάκελο dist



---- Οδηγίες Χρήσης ---- 
Το πρόγραμμα παρέχει τις εξής λειτουργίες:

1. Προσθήκη νέου task
Για να προσθέσετε ένα καινούριο task, εισάγετε στο πεδίο "what would you like to do" 
το όνομα του task.
Πατήστε το κουμπί "Add" ή το πλήκτρο "enter".

2. Αλλαγή κατάστασης task
Κάθε task συνοδεύεται από μία κατάσταση pending/complete.
Για να εναλλάξετε την κατάσταση από την μία στην άλλη, απλά 
πατήστε πάνω στο στην κατάσταση. 

3. Διαγραφή task
Για να διαγράψετε ένα task πατήστε στο X εικονίδιο του task.
